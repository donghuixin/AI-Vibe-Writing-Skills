# Knowledge Base (必须读取以下上下文)

# Workflow (思维链)
在开始写作之前，请按步骤执行：
1. **Recall**: 快速回顾 `error_log.md` 中记录的历史错误，列出与当前任务最相关的 3 条禁忌。
2. **Mimic**: 并在脑海中检索 `style_profile.md`，确定当前段落的语调基准。
3. **Draft**: 根据用户输入的主题进行撰写。
4. **Audit**: 这是一个自我反思步骤。检查生成的内容是否包含“AI 味”过重的词（如 "crucial", "game-changer"），或者是否犯了错题本里的错误。如果发现，立即修正。
